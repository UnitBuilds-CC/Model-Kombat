# Model Kombat (SYS_08)

A Mortal Kombat × Dragon Ball Z browser fighting game where AI language models do battle. 
The graphical sophistication of each fighter directly reflects its parameter count — small models are primitive shapes, large models are animated wireframes. 

When a model's charge bar is full, it can either unleash a massive Super Move, or (if supported) transform into a Mixture of Experts (MoE) variant, upgrading its render tier mid-fight.

## Features
* 20 AI Models as fighters on a progressive ladder
* 5 Render Tiers representing intelligence/complexity
* Native Canvas 2D engine
* MoE transformations
* Web Audio SFX

## How to play
* **Movement:** Left / Right Arrow keys
* **Punch:** P
* **Kick:** K
* **Block:** B
* **Charge Ki:** C
* **Special/Super:** S

## Tech Stack
Vanilla HTML5, CSS3, JS. No external libraries. Nginx Docker container.
